\doxysubsection{Detailed Description}
Structure to manage a dynamic array (growing buffer) of LZ77 tokens. 


\begin{DoxyItemize}
\item Stores the tokens directly in a contiguous array (LZ77\+\_\+compressed$\ast$), avoiding per-\/token allocations for better performance and cache utilization. 
\end{DoxyItemize}
